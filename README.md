# Sistema-de-reserva-de-vuelos
Este proyecto consiste en el desarrollo de un sistema de reservas de vuelos, cuyo objetivo principal es simular el proceso de compra y gestión de tiquetes aéreos. El sistema permite registrar usuarios, consultar vuelos disponibles y realizar reservas según la categoría del asiento elegido.
El avión cuenta con tres zonas de asientos: económica, normal y premium, cada una con un precio diferente calculado sobre una tarifa base. Al momento de reservar, el sistema verifica la disponibilidad del asiento, lo ocupa y genera una reserva con los datos del pasajero, el destino, el vuelo y el precio final correspondiente.
Además, el proyecto incluye una visualización gráfica del mapa del avión, donde se puede identificar fácilmente qué asientos están disponibles y cuáles ya han sido ocupados mediante indicadores visuales de color.
Este programa fue desarrollado como parte de un ejercicio de programación orientada a objetos, con el fin de aplicar conceptos como clases, herencia, encapsulamiento y polimorfismo. También busca reforzar habilidades de organización lógica, diseño de sistemas y modelado de situaciones del mundo real.
